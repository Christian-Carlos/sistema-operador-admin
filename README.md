# Sistema Operador Admin

## Descrição
Painel administrativo desenvolvido com Google Apps Script para consulta, gestão e auditoria de cadastros. O sistema possui controle de acesso por perfil, autenticação de operadores e administradores, além de recursos para acompanhamento e atualização de informações.

---

## Funcionalidades
- Login de operadores e administradores
- Controle de acesso por perfil
- Consulta de cadastros
- Filtros por dados do cadastro
- Edição controlada de informações
- Registro de auditoria
- Gestão de operadores
- Troca obrigatória de senha
- Visualização de histórico de ações

---

## Tecnologias
- Google Apps Script
- JavaScript
- HTML
- CSS
- Google Sheets

---

## Estrutura do Projeto

### Backend — Code.gs
Responsável pela autenticação, leitura e atualização da base de dados, controle de permissões, registro de logs e comunicação com a planilha.

### Frontend — index.html
Interface administrativa com login, painel de consulta, filtros, ficha do cadastro, edição de dados e telas de auditoria.

---

## Arquivos

## Diferenciais
- Controle de acesso por nível de usuário
